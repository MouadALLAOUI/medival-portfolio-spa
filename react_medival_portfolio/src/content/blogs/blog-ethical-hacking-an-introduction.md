# Ethical Hacking: An Introduction

Hacking isn't always criminal. Ethical hackers break into systems with permission, find vulnerabilities, and help organizations fix them before the bad guys do. It's a career, a skill set, and a mindset.

## What Is Ethical Hacking?

Ethical hacking (also called penetration testing or "pen testing") is the authorized practice of bypassing system security to identify weaknesses. The key word is **authorized** — ethical hackers have explicit permission.

They think like attackers but act like consultants. The goal is to improve security, not exploit it.

## Penetration Testing

A penetration test follows a structured process:

1. **Reconnaissance** — gather information about the target
2. **Scanning** — identify open ports, services, vulnerabilities
3. **Exploitation** — attempt to break in using discovered weaknesses
4. **Post-exploitation** — what can be accessed once inside
5. **Reporting** — document everything and recommend fixes

Each phase builds on the last. The report is the most important deliverable — it's what the client actually pays for.

## Bug Bounties

Many companies offer rewards for finding and reporting vulnerabilities. Platforms like HackerOne and Bugcrowd connect researchers with organizations willing to pay for responsible disclosure.

You can earn anything from a few hundred to tens of thousands of dollars for a single bug. Some researchers make a full-time living from bug bounties.

## Legality

Without permission, hacking is illegal — full stop. The Computer Fraud and Abuse Act (CFAA) in the US and similar laws worldwide criminalize unauthorized access.

Always get written authorization before testing any system. Bug bounty programs provide that authorization for their scope.

## Career Path

Ethical hacking is a growing field. Common certifications:
- **CEH** (Certified Ethical Hacker)
- **OSCP** (Offensive Security Certified Professional)
- **CompTIA Security+** (foundational)

Start with networking fundamentals, learn Linux, practice on legal platforms like Hack The Box or TryHackMe, then pursue certifications.

## Skills You Need

- Networking (TCP/IP, DNS, HTTP)
- Linux command line
- Scripting (Python, Bash)
- Understanding of web technologies
- Problem-solving and patience
- Strong ethics and integrity

Ethical hacking is as much about mindset as technical skill. Curiosity and persistence matter more than raw intelligence.
