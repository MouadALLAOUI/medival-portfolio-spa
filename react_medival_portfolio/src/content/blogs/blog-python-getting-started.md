# Python: Getting Started

Python is one of the most popular programming languages in the world, and for good reason. It's beginner-friendly, versatile, and used in everything from web development to data science and AI. If you're just starting your coding journey, Python is a fantastic place to begin.

## What is Python?

Python is a high-level, interpreted programming language created by Guido van Rossum in 1991. Unlike lower-level languages, Python reads almost like English, which makes it easy to learn and write. It's used by companies like Google, Netflix, and Instagram.

## Installing Python

Head over to [python.org](https://python.org) and download the latest version for your operating system. During installation, make sure to check the box that says **"Add Python to PATH"** — this saves you a lot of headaches later.

```bash
# Verify installation in your terminal
python --version
```

## The Python REPL

Python comes with a Read-Eval-Print Loop (REPL) — an interactive shell where you can type Python code and see results instantly. It's great for quick experiments.

```python
>>> 2 + 2
4
>>> "hello".upper()
'HELLO'
```

## Your First Program

Create a file called `hello.py` and add this line:

```python
print("Hello, World!")
```

Run it from your terminal:

```bash
python hello.py
```

You should see `Hello, World!` printed to the screen. Congratulations — you just wrote your first Python program!

## Comments in Python

Comments help you (and others) understand your code. Python ignores them when running the program.

```python
# This is a single-line comment

"""
This is a
multi-line comment (technically a docstring)
"""
```

## Why Python?

Python has a huge community, tons of libraries, and is used in almost every tech field. Whether you want to build websites, automate tasks, analyze data, or create AI models, Python has you covered. Its gentle learning curve means you spend less time fighting syntax and more time solving problems.

> **💡 Tip:** Practice by typing code directly into the Python REPL. Experimenting freely is one of the best ways to build muscle memory and confidence.
