# Formative Assessment: Tools and Techniques

Formative assessment is ongoing evaluation that monitors student learning during instruction. It provides feedback that can be used to improve teaching and learning.

## What is Formative Assessment?

Unlike summative assessment (final tests), formative assessment happens during learning:

```markdown
Formative vs Summative:
Formative: "How are we doing?"
- Happens during learning
- Provides immediate feedback
- Informs instruction adjustments
- Low or no stakes

Summative: "What did we learn?"
- Happens after learning
- Evaluates final achievement
- Grades and certification
- High stakes
```

## Quizzes and Quick Checks

Simple tools to gauge understanding:

```markdown
Effective Quiz Strategies:
- Start with review questions
- Use multiple question types
- Keep quizzes short (3-5 questions)
- Provide immediate feedback
- Use for self-assessment

Question Types:
- Multiple choice
- True/false
- Short answer
- Matching
- Fill-in-the-blank
```

## Polls and Surveys

Gather real-time feedback from students:

```markdown
Polling Tools:
- Mentimeter
- Poll Everywhere
- Kahoot! (for quizzes)
- Google Forms
- Socrative

Poll Ideas:
- "How confident do you feel about this topic?"
- "Which concept needs more review?"
- "Rate your understanding from 1-5"
- "What questions do you have?"
```

## Exit Tickets

Brief reflections at the end of class:

```markdown
Exit Ticket Prompts:
- "What did you learn today?"
- "What questions do you still have?"
- "How would you rate your understanding?"
- "What was most confusing?"
- "How will you apply this learning?"

Implementation:
- Last 5 minutes of class
- Collect on paper or digitally
- Review before next class
- Address common questions
```

## Rubrics for Assessment

Clear criteria for evaluating work:

```markdown
Rubric Components:
1. Task description
2. Assessment criteria
3. Performance levels
4. Descriptors for each level

Simple Rubric Example:
Excellent (4): Meets all criteria with depth
Good (3): Meets most criteria
Developing (2): Meets some criteria
Beginning (1): Meets few criteria
```

**Rubric Benefits:**
- Clarifies expectations
- Ensures consistent grading
- Provides specific feedback
- Helps students self-assess

## Peer Assessment

Students evaluate each other's work:

```markdown
Peer Assessment Process:
1. Establish clear criteria
2. Model the assessment process
3. Provide sentence starters
4. Allow revision time
5. Reflect on the process

Benefits:
- Develops critical thinking
- Reinforces learning
- Provides multiple perspectives
- Builds communication skills
```

## Self-Assessment

Students evaluate their own learning:

```markdown
Self-Assessment Strategies:
- Learning journals
- Confidence checks
- Goal setting worksheets
- Portfolio reflections
- Skill checklists

Self-Assessment Questions:
- What did I learn well?
- What do I need to practice?
- What strategies worked for me?
- What will I do differently next time?
```

## Observation and Anecdotal Notes

Informal assessment through observation:

```markdown
Observation Strategies:
- Circulate during work time
- Listen to group discussions
- Note participation patterns
- Record misconceptions
- Track progress over time

Note-Taking System:
- Use sticky notes or digital tools
- Record date, student, and observation
- Look for patterns over time
- Use data to inform instruction
```

## Using Assessment Data

Turn assessment into action:

```markdown
Data Analysis Steps:
1. Collect assessment results
2. Identify patterns and trends
3. Determine next instructional steps
4. Group students by need
5. Plan targeted interventions

Data Discussion Questions:
- What concepts do most students understand?
- Where are common misconceptions?
- Which students need additional support?
- How should instruction be adjusted?
```

## Creating an Assessment-Rich Classroom

Integrate assessment into daily practice:

```markdown
Assessment Integration:
- Begin lessons with pre-assessment
- Check understanding throughout
- Use informal observations
- End with reflection
- Adjust instruction based on data

Assessment Culture:
- Focus on learning, not grades
- Normalize mistakes as learning
- Encourage self-assessment
- Provide specific, actionable feedback
- Celebrate growth and progress
```

> 💡 Tip: Formative assessment should inform your teaching, not just evaluate students. Use the data you collect to adjust your instruction, provide targeted support, and celebrate student progress. The goal is improvement, not judgment.
