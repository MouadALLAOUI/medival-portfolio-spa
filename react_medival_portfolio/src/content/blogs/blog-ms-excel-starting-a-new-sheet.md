# MS Excel: Starting a New Sheet

Excel is everywhere — from personal budgets to massive corporate reports. It's the most widely used spreadsheet tool in the world, and knowing your way around it is a genuinely valuable skill. Let's start from scratch.

## The Interface Overview

When you open a new Excel workbook, you'll see:

- **Ribbon** — the toolbar at the top with tabs like Home, Insert, Data
- **Formula Bar** — shows the content of the currently selected cell
- **Sheet Tabs** — at the bottom, labeled Sheet1, Sheet2, etc.
- **Name Box** — top-left corner, shows the cell reference (like A1)

Take a minute to click around — familiarizing yourself with the layout saves time later.

## Cells, Rows, and Columns

Everything in Excel revolves around the **cell grid**:

- **Columns** are labeled with letters (A, B, C, ... Z, AA, AB, ...)
- **Rows** are numbered (1, 2, 3, ...)
- A **cell** is the intersection of a column and row (A1, B3, C10)

Click any cell — that's where you enter data. The active cell has a green border.

## Entering Data

Just start typing. Excel accepts:

- **Text** — letters, words, labels
- **Numbers** — values for calculations
- **Formulas** — always start with `=` (like `=A1+B1`)

Press **Enter** to confirm and move down, or **Tab** to confirm and move right. Press **Esc** to cancel.

## Basic Formatting

Make your data readable with formatting:

- **Bold, Italic, Underline** — Home tab buttons or Ctrl+B/I/U
- **Font Size** — dropdown in the Home tab
- **Cell Colors** — fill color bucket icon
- **Borders** — grid lines around cells
- **Number Format** — currency, percentage, date, etc.

Select cells first, then apply formatting. You can format an entire row or column by clicking the letter or number header.

## Resizing Columns and Rows

Hover over the boundary between column headers (or row numbers) until the cursor changes, then drag to resize. Or double-click to auto-fit the content:

```text
To auto-fit: Double-click the boundary between A and B
```

## Saving Your Work

- **Ctrl+S** — saves the current file
- **Save As** — lets you choose the location and format

Excel files use the `.xlsx` extension. You can also save as `.csv` for plain data or `.pdf` for sharing.

## Navigating Quickly

| Shortcut | Action |
|----------|--------|
| Ctrl+Home | Go to cell A1 |
| Ctrl+End | Go to last used cell |
| Ctrl+Arrow | Jump to edge of data |
| Ctrl+Space | Select entire column |
| Shift+Space | Select entire row |

> **💡 Tip:** Learn a few keyboard shortcuts early. Ctrl+Z (undo), Ctrl+C/V (copy/paste), and Ctrl+S (save) alone will save you hours over time.
