# Figma: UI/UX Design Basics

Figma is the design tool that runs in your browser and has taken the UI/UX world by storm. Whether you're designing an app, a website, or just want to understand how interfaces work, Figma is where to start.

## What Is Figma?

Figma is a collaborative interface design tool. You design in your browser, and anyone with the link can view or edit in real time. No exporting files, no "which version is final?" headaches.

It's free for personal use and has become the industry standard for UI design.

## Frames and Artboards

Everything in Figma is built inside **Frames** (previously called artboards). A frame is like a canvas for a single screen or component.

Create a frame, set its dimensions (iPhone, desktop, custom), and start designing inside it. Each frame represents one screen or state of your interface.

## Components and Reuse

**Components** are reusable design elements. Create a button once, turn it into a component, and use it everywhere. Change the component, and every instance updates automatically.

This is how design systems work — consistency across hundreds of screens without manual updates.

## Auto Layout

Auto Layout makes frames responsive. Instead of manually positioning elements, you define spacing and alignment rules, and Figma arranges everything automatically.

It's like CSS Flexbox but visual. Add padding, set gaps between items, and elements resize and reflow as content changes.

## Prototyping

Figma lets you create interactive prototypes. Link frames together, define interactions (click, hover, drag), and set transitions. Then hit **Present** to walk through your design as if it were a real app.

This is invaluable for testing flows and getting feedback before writing any code.

## Collaboration

Share your Figma file with a link. Teammates can comment directly on the design, suggest changes, and even edit simultaneously.

The commenting system is built in — no need for external tools. Leave feedback right where it matters.

> 💡 Tip: Use `Ctrl+D` to duplicate objects and `Alt+Click` to quickly measure the distance between two elements. These small shortcuts save you hours over time.
